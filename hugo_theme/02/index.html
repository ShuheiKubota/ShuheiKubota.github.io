思うところがあり、今回は list.html で代用します。

<!DOCTYPE html>
<html lang="{{ .Site.Language.Lang }}">
<head>
    <meta charset="UTF-8">
    <title>{{ .Page.Title }} - {{ .Site.Title }}</title>
</head>
<body>
    {{ .Page.Content }}

    <hr />

    <h1>{{ .Page.Title }}での .Data.Pages</h1>
    <ul>
    {{ range .Data.Pages }}
    <li><a href="{{ .Permalink }}">{{ .Title }}</a></li>
    {{ end }}
    </ul>

    <h1>.Site.Pages</h1>
    <ul>
    {{ range .Site.Pages }}
    <li><a href="{{ .Permalink }}">{{ .Title }}</a></li>
    {{ end }}
    </ul>

</body>
</html>

比較したかったので、.Site.Pages も出力するようにしています。
