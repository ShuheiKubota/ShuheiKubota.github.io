トップページのレイアウト定義をいじります。

<!--more-->

## このレイアウトの方針

- セクションを一覧表示します。

あまり凝ったことをしていませんが、普通のサイト向けを見据えるのであればちゃんと考えたが良いです。

derty では、機能の大半は `baseof.html` に寄せています。

## `home.html`

```go-html-template {linenos=table,hl_lines="9"}
{{/* トップページ */}}
{{ define "main" }}
  <h1>{{ .Page.Title }}</h1>
  {{ .Page.Content }}

  <div class="list_container">
    <ul>
      {{/* セクションが増えてきたら調整するかも… */}}
      {{ range (.Paginate .Sections.ByTitle 100).Pages -}}
        {{ partial "page_as_listitem.html" . }}
      {{ end }}
    </ul>
    {{ partial "pagination.html" . }}
  </div>
{{ end }}
{{/* vim: set et ft=html sts=2 sw=2 ts=2 : */}}
```

### セクションの一覧

いまのところページネーションするほどのセクションにはなっていませんが、将来的にはここら辺も考える必要があると思います。

`.Paginate` のところにある `100` を省略した場合、プロジェクトの設定が使われます。  
ここでは将来のことを考えてか考えずか、セクションページやタグページとは異なる件数の表示をするため、あえて大きな数を指定しています。
