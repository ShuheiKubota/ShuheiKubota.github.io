Keychron K7 を JIS 配列のキーボードとして使っている場合に便利なキーマッピングを追加するアプリ。

<!--more-->

# ダウンロード

[ダウンロード](https://github.com/shu-ahk/k7jis/releases)

# 用途

K7 に限らず US 配列のキーボードを JIS 配列として認識させている場合、いくつかのキーが使えなくなります。

k7jis は、使えなくなったキーを補完するようなキーマッピングを追加します。
また、私が個人的に若干使いやすくするためのキーマッピングも追加します。

# キーマッピング

## 共通のキーマッピング

| 入力 | 入力されたことになるキー |
|-|-|
| `Alt`+`^`         | `\`       |
| `Shift`+`Alt`+`^` | `\|`      |
| `Alt`+`/`         | `_`       |
| `Ctrl`+`/`        | `Menu`    |
| `Alt`+`Home`      | `End`     |
| `Alt`+`Esc`       | `全角半角` |
| | |
| `Alt`+`BackSpace` | Fモード(後述) |

## Fモードでのキーマッピング

| 入力 | 入力されたことになるキー |
|-|-|
| `1` | `F1`  |
| `2` | `F2`  |
|  :  |  :    |
| `9` | `F9`  |
| `0` | `F10` |
| `-` | `F11` |
| `^` | `F12` |
| | |
| `Alt`+`1` | `1` |
| `Alt`+`2` | `2` |
|  :        |  :  |
| `Alt`+`9` | `9` |
| `Alt`+`0` | `0` |
| `Alt`+`-` | `-` |
| `Alt`+`^` | `^` |
| | |
| `Alt`+`↑` | `PageUp`   |
| `Alt`+`↓` | `PageDown` |
| `Alt`+`←` | `Home`     |
| `Alt`+`→` | `End`      |
| | |
| `Alt`+`BackSpace` | Fモード終了 |
